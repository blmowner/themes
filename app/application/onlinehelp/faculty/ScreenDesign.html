Screen Design

Introduction ››

The THEMES system screen has been design into the 5 sections namely section A, B, C, D and E.

Section A to contain the THEMES Header in which it contains the name of the system, the user name and Logout button

Section B to contain the THEMES Top Panel in which it contains the list of functions for the system.

Section C to contain the THEMES Left Panel in which it contains the list of sub-functions resulting from the selection in Section B.

Section D to contain the THEMES Working Area in which it shows the page for user respond resulting from the selection in Section C.

Section E to contain the THEMES Footer in which it contains the university name and its copyright.
