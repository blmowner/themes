Figure 19: New Report

Pre-condition

  1. The proposal must be approved by the Senate
  2. Supervisor/Co-Supervisor need to accept invitation first

Post-condition

  1. New progress report can be made
  2. The progress report submitted can be view at Progress Report History
  3. Wait for the supervisor to respond the monthly progress report


Steps:-

  1. System will display all necessary field that need to be field in by user
  2. System also will display list of supervisor/co-supervisor assigned
  3. Tick on the chapter of the progress report based on the discussion
  4. Then insert necessary detail regarding progress report in the field provided
  5. Add discussion date (refer Add Discussion Date)
  6. Add Attachment (refer Add Attachment)
  7. To save progress report before submitting, click Save button
  8. To submit progress report, click Submit button.
  9. The email notification also be triggered to the supervisor (refer Appendix)
