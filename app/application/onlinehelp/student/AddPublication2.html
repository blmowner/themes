Add Publication

Getting Started - Student ›› WORK COMPLETION ›› Work Completion ›› New Work Completion ››

Prerequisite

The publication has been added by the student via Manage Publication.

Steps

  1. Click on Publication button to add the publication.
  2. The system will open the Publication page. Select Select From Manage Publication to open another page to list the already added publication.
  3. Enter the searching criteria or just click the button to get the search result.
  4. Click on the ISSN number on the ISSN Search column to populate the record into the Publication detail page.
  5. Click Add button to add the publication detail into the Publication List

Next Action

  1. Add another Publication
  2. Delete Publication

Warning

System will prompt a notification message if no publication has been selected before the addition.
