Delete Work Completion

Getting Started - Student ›› WORK COMPLETION ›› Work Completion ››

Prerequisite

The Work Completion has been saved for more changes.

Steps

  1. The Work Completion status must be in Draft in Progress by Student.
  2. Tick the checkbox for which the Work Completion is going to be deleted.
  3. Click Delete Work Completion button to delete the Work Completion from the list.

Next Action

Create another New Work Completion.

Warning

System will prompt a notification message if

  1. No Work Completion has been selected before the deletion.
  2. The selected Work Completion to be deleted is already submitted to the Supervisor

Note

  1. Work Completion can only be deleted if it is not yet submitted to the Supervisor/Co-Supervisor (Edit Mode).
  2. The deletion of the Work Completion will be applied to the one with the same Reference No.
