Edit Conference

Getting Started - Student ›› WORK COMPLETION ›› Work Completion ›› New Work Completion ››


Prerequisite

The conference has been added by the student.

Steps

  1. Update the conference detail. It can be done in multiple rows.
  2. Tick the checkbox for the conference which has been updated. It can be done in multiple rows.
  3. Click the Update button to save the changes.

Next Action

The student can

  1. Add another Conference
  2. Update Conference
  3. Delete Conference

Warning

System will prompt a notification message if mandatory field indicated by the asterisk (*) is left blank before the addition.
