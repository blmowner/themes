This is the first page sight by the student to get into the THEMES system.


Figure 2: Login Page


Prerequisite

The user must have the User ID and Password registered with the THEMES system


Steps

  1. Open the Internet Browser i.e. Mozilla Firefox or Microsoft Internet Explorer

Enter this URL http://themes.msu.edu.my/ to access the system

  1. Login page will be displayed in which the user needs to provide the User ID and Password.



Warning

  1. If the provided User ID and Password are incorrect, the system will prompt an error message and the user needs to provide the right User ID and Password again.
  2. Just in case the user forgot the password, it can be reset via the Forgot Your Password.
