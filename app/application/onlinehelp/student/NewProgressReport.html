Figure 29: New Monthly Progress Report


Steps

  1. Select the Month and Date of the progress report.
  2. Tick the Content of Discussion. It can be done in multiple rows.
  3. Enter the Description of Topic or Issue Facing by Student.
  4. Add the attachment if any.
  5. Enter the Advice from Supervisor & List of Action to be taken by student
  6. Add the attachment if any.
  7. Click Save button if you need to do some more changes. Otherwise click Submit button.
