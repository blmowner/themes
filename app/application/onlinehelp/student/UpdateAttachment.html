Update Attachment

Getting Started - Student ›› MONTHLY PROGRESS REPORT ›› Monthly Progress Report ›› New Progress Report ››

Prerequisite

The attachment has been added successfully.


Steps

  1. Update the Document Description.
  2. Tick the checkbox where the attachment needs to be updated. It can be done in multiple rows.
  3. Click on the Update button to save the changes.

Next Action

You can proceed to submit the monthly progress report.

Warning

System will prompt a notification message if no checkbox has been selected to update.
