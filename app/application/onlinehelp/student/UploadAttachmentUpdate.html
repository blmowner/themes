Upload Attachment - Update

Getting Started - Student ›› MESSAGE ›› New Message ››

Figure 50: Upload Attachment

Prerequisite

The attachment has been added into the attachment list.


Steps

  1. Update the document’s description. It can be done in multiple rows.
  2. Tick the checkbox for the document’s description being updated. It can be done in multiple rows.
  3. Click on the Update button to update the changes.


Next Action

  1. Send the message
  2. Update the attachment’s description
  3. Delete the attachment
