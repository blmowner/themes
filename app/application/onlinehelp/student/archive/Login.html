Login

This is the first page sight by the student to enter the PGTMC system.



Figure 2: Login Page


Steps:-

  1. Open any Internet Browser of your choice either Mozilla Firefox or Microsoft Internet Explorer
  2. Enter the URL to access the system as follows http://postgrad.msu.edu.my/
  3. Login page will be displayed in which the valid user credential needs to be provided i.e. Username and Password.
